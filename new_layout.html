Welcome!

Hello and welcome to the first-year Chemistry Online Data Science Learning Lab Activity at Iona University for first-year students!!


This learning lab activity was created to provide an opportunity to upcoming first-year chemistry students with basic data analysis skills that can prove useful beyond their first-year classes. This lab activity is broken up into the following sections:

Introduction provides a fundamental explanation of what data analysis is about, as well as providing real-world applications of data analysis and many of the common tools used.

Data Wrangling begins guiding you about the first phase of data analysis: data cleaning. It'll guide you through opening your CSV file, discovering its data columns and cleaning any forms of outliners and anomalies to make extract meaningful information out of the dataset.

Math Functions and Data Organization discusses the why of involving mathematical and statistical equations, by providing an additional source of extracting information from a dataset. This section only covers a few math concepts often used in data analysis as well, but most of which can be automated with functions

Data Visualization is often most interesting step in data analysis, often because of the many forms of visualizations you are capable of producing. In addition to making graphs, this section will also guide you on the smaller intricacies in developing plots, such as choosing a proper graph design, naming/formatting your axis labels, and creating a regression line.

1.1 What is Data Analysis? Why is it Important?


Answering the question of "what is data analysis", can be broken up into two parts: data, and analysis. To analyze means the process of answering questions like "how?" or "why?". To answer these questions, we must use the data thats provided. Out of all the data, we can find what we need by filtering. After the dataset is filtered and in a finalized format, we can begin to analyze it. Analyzing data comes down to finding a pattern.


As stated previously, data analysis is answering a question using any data given. Being able to draw a conclusion from information and make an informed decision for the future is extremely valuable. Having the ability to look back on the past, predict the future, and how to react to it are all crucial in every possible scenerio.

1.2 Application of Data Analysis


Data analysis has plenty of applications that prove beneficial to sectors including commercial, government, financial, etc. Here are some examples to show the real world uses of data analysis.

  • Healthcare

    Data analysis is highly beneficial in the healthcare industry by preventing disease, optimizing resources, and offering better care to patients. A patients condition can be analyzed to properly diagnose and personalize treatment to a specifc patient and meet their needs. Data analytics can be extremely useful especially for finding a disease outbreak, treatment effectiveness, and improving all operations.

  • Cybersecurity

    Data analytics play a huge role in detecting and preventing cyber attacks. Data from security systems can find patterns in user behavior, network traffic, and other anomalies. Looking at the data can help businesses and other organizations to better their security and protect sensitive data.

  • Manufacturing

    Analyzing data is extremely benefecial for manufacturing by optimizing production processing, improving predictive maintenance, and enhancing product quality. Patterns can be drawn from machine performance, sensor data, and maintenance and machinery history to predict equipment failure, minimize downtime, and guarantee efficiency. Overall, data analysis leads to higher productivity and more savings.

1.3 Data Analysis Tools


Having the right tools to analyze data is essential for drawing accurate and reliable conclusions. Here are some of the most popular data analysis applciations.

  • Excel

    Microsoft Excel is one of the most well known software applications for data analysis. Excel provides spreadsheet functions to manage large datasets as well as graphing and computing tools. Excel is an extremely versatile platform and is crucial for any data analyst, hence the activities in this tutorial are based on microsoft excel data analysis tools.

  • MySQL

    MySQL is a popular open source relational database management system that stores data. Popular social media platforms such as Facebook, Twitter(X), and YouTube have used MySQL. SQL(Structured Query Language) is primarily used for relational database management systems, allowing for structured tables of relational databases.

  • Python

    Python is known as one of the most popular programming languages in the world. Python is a multi-paradigm language, therefor being used for many possible uses in software, web development, and data analysis. Python also has extensive libraries for use such as data analytic tools like Numpy and Pandas. Python is overall very useful and important to any data analyst.

Section 2 Data Wrangling

Often times when collecting data in experiments and trial runs, raw data collecting is prone to a variety of issues (such as having duplications and missing points, structural inconsistencies, unwanted outliners) that inherently makes creating visualizations a lot more difficult, and in turn, makes extracting any meaningful information much more harder.

In data analysis, it is widely understood that most data calls for some degree of manipulation for its analysis later on, this is called data cleaning. The purpose for data cleaning solves many of those issues mentioned before, increasing analysis efficiency, validity and data integration.


2.1 Dataset File Types

Datasets can come packed in a variety of file types, but most commonly they have come packaged as .csv (comma separated values) files, .xls or .xlsx (Office Open XML) and .zip (ZIP) files, though with the rise of JS libraries, newer datasets are being stored as JSON files too.

In this lab, we'll be using datasets compiled as CSV, XLSX, or JSON files to further demonstrate the capabilities of working with each file type.
